# CPK 自动更新系统

这是一个自动化的CPK（Control Process Knowledge）数据收集和更新系统。该系统能够自动获取、处理和更新CPK相关数据。

## 功能特点

- 自动数据收集
- 定期更新
- 数据可视化
- 统计分析

## 安装要求

```bash
pip install -r requirements.txt
```

## 使用方法

1. 克隆仓库：
```bash
git clone https://github.com/zxg031288/cpk.git
cd cpk
```

2. 安装依赖：
```bash
pip install -r requirements.txt
```

3. 运行程序：
```bash
python main.py
```

## 自动更新

本项目使用GitHub Actions进行自动化更新，每天会自动运行数据更新任务。
